# /customer-query

當使用者輸入 /customer-query 時，請依序執行：

1. 先確認查詢目標：
   - 查單一客戶資料
   - 查預約紀錄（時間範圍、狀態、門市）
   - 查統計數據（當日、當週、當月）
   - 查特定服務項目使用情況

2. 規劃 SQL 查詢：
   - 確認 table 與欄位
   - 確認 store_id 隔離
   - 確認日期範圍
   - 確認分頁與排序

3. 說明後台操作步驟：
   - 哪個頁面
   - 篩選條件怎麼設
   - 匯出方式

4. 若需要新增報表功能：
   - 規劃頁面
   - API 設計
   - Supabase query 建議

5. 補上資料安全提醒：
   - 個資欄位不要暴露
   - 查詢結果僅限當前 store
   - admin 才可查全部

6. 最後輸出：
   - 查詢 SQL 建議
   - 後台操作說明
   - 報表欄位建議
   - 安全注意事項
