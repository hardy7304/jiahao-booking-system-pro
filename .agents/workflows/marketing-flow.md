# /marketing-flow

當使用者輸入 /marketing-flow 時，請依序執行：

1. 先確認行銷目標：
   - 拉新客還是留舊客
   - 推播對象：所有用戶、特定門市、特定標籤
   - 行銷管道：LINE 推播、landing page、表單
   - 是否要搭配優惠碼或限時活動

2. 規劃內容：
   - 標題與主文
   - 圖片或 Flex Message 格式
   - CTA 按鈕與連結
   - 活動時間與條件

3. 規劃 n8n 自動化：
   - 觸發條件（時間、事件、表單送出）
   - 受眾篩選邏輯
   - 推播節點設定
   - 追蹤點擊或轉換

4. 規劃 lead magnet 流程：
   - 表單 URL
   - 填完後自動回 LINE 訊息
   - 記錄到 Supabase 或 Notion
   - 後續跟進 n8n 自動化

5. 規劃成效追蹤：
   - 推播人數
   - 點擊率
   - 預約轉換數
   - LINE 加好友數

6. 最後輸出：
   - 活動企劃摘要
   - n8n workflow 節點建議
   - 訊息文案草稿
   - 追蹤方式
   - 上線前確認清單
